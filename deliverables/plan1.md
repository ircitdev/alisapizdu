# План улучшений v2

## Быстрые победы (1-2 часа каждая)

### 1. Топ ответов Алисы
Вкладка/фильтр "Лучшие" — сообщения отсортированные по голосам. Люди будут шарить топовые ответы.

### 2. Счётчик просмотров на сообщении
"Это сообщение видели 1,247 человек". Социальное доказательство.

### 3. Реакции вместо лайков
Вместо 👍👎 добавить 😂🔥💀🤡 — как в Telegram. Веселее и больше вовлечения.

### 4. Звук печатания
Тихий звук клавиатуры пока Алиса "печатает" ответ. Эффект присутствия.

### 5. Конфетти при первом ответе
Анимация после получения ответа Алисы. Дофаминовый удар.

---

## Средние фичи (3-5 часов)

### 6. "Алиса дня"
Автоматически выбирать лучший ответ за сутки, закреплять наверху с бейджем. Мотивирует голосовать.

### 7. Цепочки ответов
После ответа Алисы кнопка "Переспросить" (платная, 500р) — продолжить диалог. Контент x2.

### 8. OG-превью для каждого сообщения
При шеринге ссылки `/#msg-42` в Telegram/VK показывать текст сообщения + ответ Алисы как картинку-превью. Кликабельность x5.

### 9. Telegram-бот
`/ask` в боте → то же самое, ответ Алисы + ссылка на ленту. Новый канал трафика.

### 10. Таймер до следующего вопроса
Красивый обратный отсчёт "до следующей попытки: 14:32:07" с анимацией. Создаёт предвкушение.

---

## Стратегические (1-2 дня)

### 11. Лидерборд
Таблица "кто задал больше всего вопросов" по user_id. Геймификация.

### 12. Тематические дни
Понедельник = "Алиса-философ", пятница = "Алиса-поэт" (менять системный промпт по дням). Повод возвращаться.

### 13. API для встраивания
Виджет "последний ответ Алисы" для сторонних сайтов. Бесплатная дистрибуция.
